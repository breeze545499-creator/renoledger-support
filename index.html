常见问题

Q: 数据会丢失吗？
A: 所有数据存储在您的设备本地。建议定期使用"设置 > 数据导出"功能备份到 CSV 或 PDF。如果卸载应用，数据会被删除。
Q: 如何恢复购买？
A: 在应用内点击"恢复购买"按钮，或使用相同的 Apple ID 重新下载应用，购买记录会自动恢复。
Q: 支持多设备同步吗？
A: 目前不支持自动同步。您可以通过导出/导入 CSV 文件手动转移数据。我们计划在未来版本支持 iCloud 同步。
Q: Face ID / Touch ID 无法使用？
A: 请确保在 iOS 设置中已启用 Face ID / Touch ID，并在应用"设置 > 安全锁定"中开启生物识别。
Q: 如何导出报表？
A: 进入"报表"页面，点击右上角导出按钮，选择 CSV 或 PDF 格式即可导出。
